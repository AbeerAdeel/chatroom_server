# Chatroom Server



## Commands
Run this command from the root to run and compile the api

```bash
yarn start
```
Run this command from the root to run the mocha tests
```bash
yarn test
```

## Tech Stack
* Node.js
* Socket.io
* HTML5
* CSS3
* Mocha (Testing)
